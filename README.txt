
使用说明：需要将input和output文件夹放入到README.txt所在的文件夹内，此外，如果要运行test.ipynb需要读入ckpt文件。
（一）
main文件是进行训练和验证的主体代码文件。主要内容有：  
1、定义了训练集和数据集，并加载数据。 
2、定义模型，以及优化器、梯度下降算法组合  
3、训练模型，并可视化

（二）
test文件用于测试和提交。
主要工作：
1、从chekcpoint文件夹内部读取指定的模型参数；
2、加载测试集数据；
3、在测试集上生成预测值；
4、按照kaggle上的要求生成相应的submission.csv文件  
请注意：提交的代码包中并没有ckpt文件，所以如果要运行test文件，需要先执行main文件得到ckpt文件后才可以生成submission文件，不然会报错的。
